# Cooking Assistant Chatbot

Simple Flask web app with AI-powered cooking chatbot (Gemini API).

## Run locally
1. Create venv and activate.
2. `pip install -r requirements.txt`
3. Put your `GEMINI_API_KEY` in `.env`
4. `python app.py`
5. Open `http://127.0.0.1:5000/`

## Features
- Search by recipe name / ingredient
- Ingredient filter + Veg/Vegan/Non-Veg
- Video links
- AI chatbot powered by Gemini API
